Backend Setup Instructions
Quick Setup Guide

Follow these steps to enable the full PHP/MySQL backend:

1. Prerequisites
  • PHP 8.1+ installed
  • MySQL 8.0+ running
  • Composer package manager
2. Install Dependencies
cd server
composer install
3. Database Setup
mysql -u root -p
CREATE DATABASE bits_college;
EXIT;
4. Start Backend
php artisan migrate --seed
php artisan serve --port=8000
Pro Tip: Use npm run dev to start both frontend and backend servers simultaneously!
